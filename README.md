# 🎤 AI Voice Agent

## Overview

An AI-powered Voice Assistant built using HTML, CSS, JavaScript, and n8n.

The application listens to the user's voice, converts speech to text, sends the request to an AI workflow through a webhook, receives the AI response, and reads it aloud using text-to-speech.

## Features

* 🎤 Speech Recognition
* 🤖 AI Response Generation
* 🔊 Text-to-Speech
* 💬 Conversation History
* 🌐 n8n Webhook Integration
* 📱 Responsive User Interface

## Tech Stack

* HTML
* CSS
* JavaScript
* Web Speech API
* n8n
* Google Gemini
